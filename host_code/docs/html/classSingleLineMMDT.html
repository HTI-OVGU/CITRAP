SingleLineMMDT Class Reference

Class provides functionality to do memory mapped data transfer to and from FPGA. The channel to use for the data transfer can be specified. It makes use of a single line.

Example usage :

SingleLineMMDT singleLineMMDT; // Instatiate the class variable.

typedef float dataType; // Specify the desired data type in place of float

std::vector<dataType> data = {255.23, -15.1, 429.5, 15.433}; // Create a array of data

std::string data = "test string"; // In case of string data

singleLineMMDT.sendToFPGA(data, dataSize, address, DEVICE_NAME_WRITE);. More...
